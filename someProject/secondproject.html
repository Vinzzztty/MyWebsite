User Management Project "CRUD Application"

Awal dari pembuatan Aplikasi ini adalah ketika aku berhasil menguasai sedikit tentang bahasa pemrograman untuk Front-End seperti Html, CSS, Javascript. Aku berfikir kenapa aku tidak mempelajari Back-End juga. Aku juga sebagai salah satu member di GDSC ITB tahun 2021/2022, disitu ada sebuah workshop tentang database modelling, routing, dan lain sebagainya. Aku mencoba memahami dan mempelajarinya setelah workshop tetapi tidak tahu bagaimana mengimplementasikannya. Disitulah aku coba search youtube untuk mencari tahu apa saja sih yang bisa diimplementasikan dari database tersebut, dan aku menemukan salah satu video yotuber India dan mencoba mengikuti dan mencobanya sendiri di

Sistem kerja dari aplikasi ini cukup gampang untuk dipahami seperti Menampilkan data, Menginput data baru, Mengsunting data yang sudah diinput dan juga bisa menghapus data.

Aplikasi ini juga menggunakan Node JS, Express, dan MongoDB sebagai databasenya. Code splitting juga digunakan untuk menyimpan data supaya enak dilihat dan mudah diperbaiki jika ada error.
