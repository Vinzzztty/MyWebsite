C++ Projects "Number System Conversion Calculator"

Project ini dimulai saat aku masih berada di bangku kuliah semester 1 jurusan S1 Informatika pada akhir bulan Desember 2021. Ini merupakan project pertamaku sejak pertama kali belajar bahasa pemrograman C++ dan project pertama yang dibuat bersama 5 anggota lain.

Sistem kerja kalkulator ini lumayan sederhana, kita memasukkan pilihan konversi lalu hasilnya akan langsung keluar.

Fitur penting di kalkulator ini adalah Validasinya, yaitu bisa mengetahui apakah user benar benar memasukkan angka bilangan oktal, biner, desimal ataupun hexadesimal sesuai ASCII.

Pada project pembuatan "kalkulator sistem konversi bilangan" aku berperan sebagai :

  • Pembuat fungsi konversi Oktal ke Biner
  • Pembuat fungsi konversi Hexadesimal ke Oktal

    • Fitur dalam Kalkulator Konversi Bilangan:
    • - Konversi Biner ke Oktal
    • - Konversi Biner ke Desimal
    • - Konversi Biner ke Hexadesimal

    • - Konversi Oktal ke Biner
    • - Konversi Oktal ke Desimal
    • - Konversi Oktal ke Hexadesimal

    • - Konversi Desimal ke Biner
    • - Konversi Desimal ke Oktal
    • - Konversi Desimal ke Hexadesimal

    • - Konversi Hexadesimal ke Biner
    • - Konversi Hexadesimal ke Oktal
    • - Konversi Hexadesimal ke Desimal
